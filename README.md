## ✨ Funcionalidades Principais

### 🎨 **Interface & Design**
- **Design Responsivo Premium**: Layout adaptativo que funciona perfeitamente em todos os dispositivos
- **Animações Avançadas**: Transições suaves e efeitos visuais elaborados usando Framer Motion
- **Tema Dark Moderno**: Interface elegante com gradientes personalizados e efeitos de glassmorphism
- **Rede Neural Animada**: Background interativo com nós conectados simulando uma rede neural

- **Server-Side Rendering (SSR)**: Carregamento otimizado com Next.js 15

- **Performance Otimizada**: Lazy loading de imagens e componentes
- **SEO Friendly**: Meta tags otimizadas e estrutura semântica
- **TypeScript**: Tipagem estática para maior confiabilidade do código
- **Animações Performáticas**: Uso de CSS transforms e Framer Motion para animações 60fps

## 🚀 Instalação e Execução

### **Pré-requisitos**
- Node.js 18.x ou superior
- npm ou yarn

### **1. Clone o repositório**
```bash
git clone https://github.com/AlvaroHenriqueOP/portifolio_alvaro.git
cd portifolio_alvaro
```

### **2. Instale as dependências**
```bash
npm install
# ou
yarn install
```

### **3. Execute o projeto**
```bash
# Modo desenvolvimento (com Turbopack)
npm run dev
# ou
yarn dev

# Build para produção
npm run build
# ou
yarn build

# Iniciar versão de produção
npm start
# ou
yarn start
```

### **4. Acesse o projeto**
Abra seu navegador em `http://localhost:3000`
